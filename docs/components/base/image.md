

# image 图片

图片容器，保留原生 img 的一些特性

## 基础用法
:::demo
image/base
:::

## 设置宽高

通过设置`width`、`height`属性改变大小

:::demo
image/size
:::

## 设置头像

通过设置`round`属性

**`scale`属性和`round`属性一起使用时，`scale`不会生效**
:::demo
image/avatar
:::
## 缩放

鼠标移入缩放效果，通过设置`scale`属性

**`scale`属性和`round`属性一起使用时，`scale`不会生效**
:::demo
image/scale
:::
