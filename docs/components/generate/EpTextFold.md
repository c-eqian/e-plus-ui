# EpTextFold
## Parameter 参数
| 参数 | 类型 | 可选 | 描述 | 默认值 |
| :-------: | :-------: | :-------: | :-------: | :-------: |
| isFold | `boolean` | `false` | 是否显示展开/折叠 | -|
| line | `number` | `false` | 超过几行显示折叠 | `3`|
| position | `left \| right` | `false` | 折叠按钮显示位置 | `right`|
## Slots 插槽
|    插槽名    |  描述   |
|:---------:|:-----:|
| `default` | - |
| `expand` | - |
