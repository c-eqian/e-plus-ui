# Icon  

 ::: tip 提示
类型定义说明通过自动生成
:::

## IconProps 



### prefix
前缀


- **Type:** `string`


- **Default:** `icon`

### name
名称


- **Type:** `string`


### color
颜色


- **Type:** `string`


- **Default:** `currentColor`

### width
宽 如果是不带单位，默认单位为px


- **Type:** `string | number`


- **Default:** `1rem`

### height
高 如果是不带单位，默认单位为px


- **Type:** `string | number`


- **Default:** `1rem`
