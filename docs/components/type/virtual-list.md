# Virtual-List API 

 ::: tip 提示

## IVirtualList 



### className
自定义类名


- **Type:** `string`


### height
容器高度，计算显示数量


- **Type:** `number`


- **Default:** `500`

### width
宽度


- **Type:** `number | string`


- **Default:** `100%`

### itemHeight
预估每行高度


- **Type:** `number`


- **Default:** `30`

### isDynamic
是否动态高度


- **Type:** `boolean`


- **Default:** `false`

### cache
缓存数量


- **Type:** `number`


- **Default:** `2`

### data
数据


- **Type:** `T[]`


- **Default:** `[]`
