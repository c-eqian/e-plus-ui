`EPlus-UI`，一个基于`element-plus`和`tailwindcss`的组件库，`EPlus-UI`旨在消除冗余代码，通过简单配置即可生成业务组件，让开发过程变得更为高效、灵活。

# 开发
交互式创建组件模板
## 本地安装
项目根目录下运行
```shell
pnpm run build:create
```
## 链接
项目根目录下运行
```shell
npm link ./create-template
```
## 创建模板
项目根目录下运行, `button`为组件名称，`e-plus-ui`为`ci`名称
```shell
e-plus-ui button
```
